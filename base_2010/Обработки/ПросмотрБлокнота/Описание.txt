	Персональный блокнот позволяет формировать список сообщений, которые выводятся
при работе с программой в заданное пользователем время и с определенной периодичностью

Список сообщений уже введенных в систему содержит:
-	дату очередного напоминания;
-	время очередного напоминания;
-	периодичность напоминания;
-	краткий комментарий

Периодичность напоминания может быть:
(Р)	- 	для одноразовых сообщений;
(Д)	- 	для ежедневных сообщений;
(Н)	- 	для еженедельных сообщений;
(М)	- 	для ежемесячных сообщений;
(К)	- 	для ежеквртальных сообщений;


	Для ввода новой записи в блокнот (нового сообщения) нажмите кнопку "Новое"
Для редактирования записи блокнота дважды щелкните на ней мышью или 
выберите ее курсором, а затем нажмите кнопку "Изменить"
Для удаления  записи блокнота выберите ее курсором, а затем нажмите кнопку
"Удалить"

	После редактирования списка сообщений нажмите кнопку "Закрыть"

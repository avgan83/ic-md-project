Персональный блокнот позволяет формировать список сообщений, которые выводятся
при работе с программой в заданное пользователем время и с определенной периодичностью.

Комментарий напоминания кратко отражает суть сообщения. Собственно текст сообщения 
вводится в многострочном поле ввода.
Для вывода сообщения с определенной периодичностью необходимо установить флажок
"Периодическое" и выбрать одно из значений: ежедневно, еженедельно, ежемесячно, 
ежеквартально.

Дата вывода сообщения интерпретируется для разных типов периодичности по-разному:
-     Для одноразового (непериодического) сообщения это собственно дата вывода сообщения
-	Для ежедневного сообщения дата является просто датой отсчета, начиная с которой 
	будет ежедневно выводится окно предупреждения.
-	Для еженедельного сообщения предупреждающее окно будет выводится в заданную дату и 
	повторяться еженедельно в тот день недели, на который приходится заданная дата.
-	Для ежемесячного сообщения предупреждающее окно будет появляться каждый раз, 
	когда число текущей даты совпадает с числом даты сообщения.
    
Когда при работе с программой на экране появляется окно предупреждения о событии 
можно просто его закрыть или предварительно установить флажок "Повторить еще"

Это приведет к тому, что через заданное время (поле ввода "через") данное окно 
предупреждения будет выведено повторно.

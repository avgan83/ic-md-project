Документ "Льготы" служит для начисления льгот по подоходному налогу в разрезе сотрудников.
Последовательность заполнение документа:
1. Проставляем дату документа (изначально датой документа является рабочая дата);
ВНИМАНИЕ:
Документ "Льготы" должен быть сформирован и проведен первого числа ка-ждого месяца первым из
всех документов по зарплате и только единожды. 
2. Возможно два варианта заполнения данного документа:
- первый: путем ручного выбора соответствующего сотрудника из справочника. Сумма льготы
будет автоматически проставлена исходя из данных указанных в истории сотрудника на дату
формирования документа.
- второй: путем автоматического заполнения сотрудников. Для этого необходимо нажать на кнопку
"Заполнить" - в результате в табличную часть документа попадут все сотрудники у которых исходя
из данных указанных в их истории существует льгота на дату формирования документа.
ВНИМАНИЕ:
1. Изменение суммы льготы по сотруднику невозможно из данного документа. Корректировка
льготы доступна только в справочнике "Сотрудники".
2. Льготы не начисляются на сотрудников которые были приняты на работу в этом месяце;
3. В случае ручного заполнения строк программа автоматически предложит заполнение следующей
строки путем открытия справочника "Сотрудники" для выбора новой аналитики. При наличии 
дополнительных позиций ввод можно продолжить, при их отсутствии нажать на кнопку "Закрыть"
в нижней части экрана, либо клавиши "Esc" - справочник будет закрыт и вы вернетесь в документ. 
После чего надо будет удалить зарезервированную дополнительную строчку в документе
содержащую текущий номер строки без какой-либо дополнительной информации. Это 
осуществляется нажатием клавиши "Esc".
Печать:
Получение печатной копии документа осуществляется нажатием на кнопку "Печать". 
Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем
"Да" для записи его в базу и "Нет" для отказа. При положительном ответе появится вопрос
"Провести документ?" - ответ "Да" сформирует по документу проводки, ответ "Нет" - проводки
формироваться не будут, но документ будет сохранен в базе и при необходимости к нему можно
будет вернуться.

Кнопка "Закрыть" предложит вам сохранить документ без формирования проводок по нему.

Документ предназначен для формирования бухгалтерских проводок по закупкам, прилагая печатную форму закупочного акта.
Порядок заполнения документа:
1. Проставляем дату документа (изначально датой документа является рабочая дата);
2. Заполняем реквизиты поставщика: счет поставщика (по умолчанию счет 539.3 "Краткосрочные обязательства другим кредиторам") и аналитику из справочника "Контрагенты". Если ведется аналитика в разрезе договоров, в следующем окошке выбираем договор;
3. Проставляем серию и номер закупочного акта, на основании которого оформляется документ;
4. Проставляем  счета подотчетного лица: синтетический и аналитические данные;
5. Выбираем вид налога, удержанного при закупке. В следующем окошке будет автоматически рассчитываться сумма подоходного налога, удержанная у источника выплат. При необходимости ее корректировки необходимо поставить флажок "Вручную";
6. При установке флажка "Формирование проводок по подотчетному лицу" документ автоматически сформирует проводки по закрытию подотчетных сумм;
7. Заполняем табличную часть перечнем закупленных ТМЦ у поставщика, указывая счет ТМЦ, аналитику и склад оприходования, а также количество закупленных ТМЦ и сумму по ним;
8. Нажатие клавиши "Enter" является подтверждением правильности вноса информации по строке;
9. По окончании заполнения строки программа автоматически предложит заполнение следующей путем резервирования новой строки. При наличии дополнительных позиций ввод  можно продолжить, при их отсутствии нажать на клавишу "Esc" - зарезервированная дополнительная строка в документе содержащая текущий номер строки без какой-либо дополнительной информации удалится.

Печать:
Получение печатной копии документа осуществляется нажатием на кнопку "Печать". 

Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем "Да" для записи его в базу и "Нет" для отказа. 
При положительном ответе появится вопрос "Провести документ?" - ответ "Да" сформирует по документу проводки, 
ответ "Нет" - проводки формироваться не будут, но документ будет сохранен в журнале и при необходимости к нему можно будет вернуться.
Кнопка "Закрыть" предложит Вам сохранить документ без формирования проводок по нему.

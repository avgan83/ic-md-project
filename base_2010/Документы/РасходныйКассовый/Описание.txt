Последовательность заполнение документа:
1. Нумерация расходного кассового номера ведется автоматически. При необходимости нумерации
документа не с единицы (приходные кассовые ордера уже заполнялись вручную) можно ввести
следующий порядковый номер. Дальнейшая нумерация будет продолжена с введенного вами номера;
ВНИМАНИЕ: 
Нумерация приходных и расходных кассовых ордеров ведется параллельно. Ведение кассовых
ордеров в леях и в валюте также ведется параллельно;
2. Проставляем дату документа (изначально датой документа является рабочая дата);
3. Выбираем счет кассы (по умолчанию - 241.1);
4. В ячейке "Денежные средства" выбираем необходимую аналитику по движению данного
денежного средства.  
ВНИМАНИЕ: 
При корреспонденции двух счетом 24 группы (241, 242, 245, 246 и др.) выбираем аналитику
"Движение по группе 24". Это необходимо для корректного заполнения "Отчета о движении
денежных средств", поскольку данные обороты в него не должны попадать;
5. В разделе "Реквизиты получателя" вносим корреспондирующий с кассой счет, а в ячейку
"Получатель" - получателя из соответствующего справочника. Узнать о взаимосвязи счетов и 
аналитики (субконто) можно просмотрев "План бухгалтерских счетов по НСБУ" находящийся в
программе. 
В случае выбора счета связанного со справочником "Контрагенты" появляется дополнительное
поле "Договор" для конкретизации договора на основании которого был осуществлен расход денег
из кассы.
ВНИМАНИЕ: 
При корреспонденции счета 24 группы (242, 245, 246 и др.) в ячейку получателя вносится
аналитика
"Движение по группе 24" из справочника "Денежные средства". Это необходимо для корректного
заполнения "Отчета о движении денежных средств", поскольку данные обороты в него не должны 
попадать;
6. В поле "Выдано" вносится фамилия и инициалы материально ответственного лица -
получателя;
7. В ячейку "Основания" вносится комментарий объясняющий выбытие денежных средств из
кассы, в ячейку "Приложение" - документы прилагаемые к данному кассовому ордеру;
8. Проставляется сумма расхода. В случае, когда сумма расхода включает НДС необходимо
проставить галочку в соответствующее окошко "НДС";
9. При выбытии денежных средств по оплате оказанных услуг с которых удерживается налог 5%
необходимо проставить галочку в окошке "Услуга 5%". Далее необходимо конкретизировать 
аналитику налога и при необходимости выбрать счет налога (по умолчанию 534.11). ). В ячейке
"Сумма налога" будет автоматически рассчитан 5% налог с суммы введенной в ячейку "Сумма".
Для контроля за правильностью заполнения этих полей существует ячейка "Сумма уплаченного
дохода" в которой будет автоматически рассчитана сумма с которой осуществлено
налогообложение. 
ВНИМАНИЕ: 
Внизу окна документа есть поле "Формировать проводки" для возможности сохранения
документа без формирования проводок либо с их формированием. Наличие галочки в данном 
окошке вызывает формирование проводок по кассовому ордеру, отсутствие - нет.Эта 
возможность необходима для исключения дублирования различных документов формирующих
одни и те же проводки. Так, например, при инкассации денег из кассы на расчетный счет данная
операция пройдет по двум документам: выписке банка и расходному кассовому ордеру.В этом
случае можно сформировать проводку документом "Банковская выписка", а документ
"Расходный кассовый ордер" использовать только для получения печатной формы документа
без формирования по нему проводок. 
Печать:
Получение печатной копии документа осуществляется нажатием на кнопку "Печать". 
Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем
"Да" для записи его в базу и "Нет" для отказа. При положительном ответе появится вопрос
"Провести документ?" - ответ "Да" сформирует по документу проводки, ответ "Нет" - проводки
формироваться не будут, но документ будет сохранен в базе и при необходимости к нему можно
будет вернуться.
Кнопка "Закрыть" предложит вам сохранить документ без формирования проводок по нему.

	Это универсальный, т.е. работающий в любой конфигурации, 
отчет. Он позволяет просматривать и выводить на печать любой из 
существующих в конфигурации журналов документов, а также 
добавлять произвольное количество новых журналов с возможностью 
настройки состава документов, входящих в журнал, отбора документов 
по условиям и в интервале дат, настройки колонок и их 
упорядочивания, и возможностью получения итогов по колонкам.
	Если нужно посмотреть / распечатать стандартный 
(определенный в данной конфигурации) журнал, то необходимо при 
помощи соответствующих элементов управления на любой закладке 
выбрать вид журнала, выбрать интервал на закладке "Общие" (если 
требуется выбрать документы за весь период работы системы, то 
необходимо отключить флажок "Ограничить интервалом") и нажать 
кнопку "Сформировать" или щелкнуть два раза мышью  в левом верхнем 
углу табличной части отчета. Пустая начальная дата при вкл. флажке 
означает, что выборка будет производиться начиная с самого первого 
существующего в базе данных документа. Аналогично, пустая конечная 
дата при вкл. флажке означает, что выборка будет производиться по 
самый последний существующий в базе данных документ.
	После того как нужный журнал выбран, в закладке "Состав" 
появится список документов, а  в закладке "Графы" заполнится таблица 
колонок (граф), выводимых в выбранном журнале в соответствии с 
настройками данной конфигурации. Нажав кнопку "Состав", в закладке 
"Состав", можно отредактировать список видов документов выводимых 
в журнале. Нажав кнопку "Добавить", в закладке "Графы" появиться 
окно диалога для настройки новой графы. В нем необходимо, задать имя 
графы (оно будет показываться в качестве заголовка колонки в журнале), 
а также определить для каждого (из выбранных в закладке "Состав") 
документа - реквизит, который будет выводиться для данного документа, 
в таблице отчета в данной колонке (графе). Выбор производиться 
двойным щелчком мыши в соответствующей ячейке таблицы и 
последующим выбором нужного реквизита из предложенного списка. 
Для установки пустого значения необходимо выбрать последнее 
значение из списка - "Очистить". Для выбранных реквизитов табличной 
части документа (таких как "Сумма", "НДС", "НП" в типовой 
конфигурации) в отчет будет выводиться итог (сумма) по всем строкам 
документа. Кроме того существует возможность в качестве возможных 
значений графы для определенных видов документов определить 
выражение из числовых реквизитов данного документа (например для 
торговых накладных в типовой конфигурации это могут быть выражения 
вида: (Сумма - НДС) / Курс, или (Сумма + НП) * Курс и т.д.).
	Диалог для настройки графы используется как при создании 
новой, так и при редактировании существующей графы. Для перехода в 
режим редактирования существующей графы необходимо установить 
курсор на нужную строку в таблице граф и нажать кнопку "Изменить" 
или установить курсор в нужной строке на имени графы и нажать "Enter" 
или по двойному щелчку мыши на имени графы. После получения 
отчета, вызвать диалог редактирования свойств графы можно щелкнув 
два раза мышью в строке заголовков отчета на наименовании конкретной 
графы. Графы можно включать или исключать из отчета, изменяя 
колонку "+" ("+" - показывать колонку). Включив настройку 
упорядочивания (флажок), появляется возможность задать порядок 
упорядочивания (по возрастанию / убыванию) для каждой графы 
журнала. Порядок вывода колонок в отчете и порядок их упорядочивания 
(при включенной настройке) можно менять располагая графы в таблице в 
нужной последовательности используя кнопки "Вниз", "Вверх". В 
таблице граф можно также выбрать вид итога для числовых колонок 
(сумма, среднее, максимум, минимум) и задать ширину колонки в 
символах. Чтобы отредактировать свойства графы в таблице, необходимо 
щелкнуть два раза по нужной колонке и в появившемся списке / диалоге 
выбрать нужное значение.	
	Имеется возможность задавать условия отбора документов. Для 
задания условий необходимо перейти на закладку "Условия". По кнопке 
"Добавить" -появляется диалог для выбора условий, предварительно 
система выдаст запрос - "Для чего определяется условие". Условия 
можно задавать как для всех документов по значению определенной 
графы ("для графы: xxxx"), так и для отдельных видов документов по 
значению реквизита определенного документа. В диалоге необходимо 
выбрать нужный атрибут в поле "Данные", проставить условие и в поле 
"Значение" определить нужное значение отбора. Если условие должно 
использоваться при формировании выборки документов, то нужно 
проставить признак "Использовать условие". Для  добавления условия в 
таблицу формы отчета, нажать на кнопку "ОК", для 
отмены выбора - кнопка "Отмена".
	Добавленные условия можно включать или исключать из 
условий формирования выборки документов, изменяя колонку "+" ("+" - 
включить в запрос). Чтобы отредактировать условие в таблице, 
необходимо щелкнуть два раза по нужной колонке, в появившемся 
диалоге условий изменить параметры и нажать на кнопку "ОК".
	Кроме того, есть возможность производить выборку только 
помеченных / непомеченных на удаление документов (или всех) и для 
проведенных / непроведенных документов (или всех). Для этого нужно 
поставить переключатель на закладке "Общие" в соответствующее 
положение (см. руководство пользователя).
	Как было сказано выше, списком журналов тоже можно 
управлять. Кнопка "Очистить" служит для удаления всех журналов из 
списка. Кнопка "Удалить" - удаляет текущий журнал в списке. При 
нажатии кнопки "Добавить", система предлагает добавить все 
стандартные журналы в список (если он пустой) или скопировать 
настройки для нового журнала из текущего. При ответе "Нет" в обоих 
случаях программа предлагает выбрать журнал из списка всех журналов 
и задать ему имя (если стандартное не устраивает или уже есть в списке).
При ответе "Да" на вопрос "скопировать настройки для нового журнала 
из текущего" программа запрашивает только имя нового журнала, а 
настройки (графы, интервал, условия отбора и т.д.) копируются из 
текущего журнала.
	Настройки элементов формы отчета сохраняются автоматически 
для каждого вида журнала. 

Замечания: 
	- использование упорядочиваний, выражений и итогов      
	  увеличивает время построения отчета.

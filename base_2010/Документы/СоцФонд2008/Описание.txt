Документ СоцФонд представляет собой пакет данных для выгрузки электронных деклараций (REV).
Документ(выгруженные данные из этого документа) отвечает всем требованиям к структуре и форматам электронных
документов (деклараций) для персонифицированного учета взносов Застрахованных Лиц (ЗЛ) .

Документ предназначен как для автоматического формирования так и для ручного ввода данных о сотрудниках за отчетный период.

Табличная часть документа СоцФонд содержит сведения о доходах Застрахованного Лица (ЗЛ) за отчетный год-период 
выбранный в документе по формам деклараций REV5,REV5a.
Поля: Период,ТКСС(территориальная касса),Ответственный-ОБЯЗАТЕЛЬНЫ для заполнения.
	   Тариф по умолчанию заполняется для случаев когда у сотрудника не указана категория страхования.

Перед тем как выгрузить данные в каталог выгрузки проверьте правильность заполнения данных о фирме:Код НКСС,
руководитель предприятия,гл. бухгалтер.
	   
Документ СоцФонд_PL содержит информацию ярлыков пачек с декларациями по форме REV2.
Документ заполняется вручную или автоматически из документа СоцФонд.

Документ СоцФонд_NA содержит анкетные данные ЗЛ по формам деклараций REV5,REV5a.
Документ заполняется вручную или автоматически из документа СоцФонд;

Документ СоцФонд_DI содержит сведения о стаже ЗЛ за отчетный год по форме деклараций REV5.
Документ заполняется исходя из информации указанной по сотруднику (обпатить внимание на периодичность)
вручную или автоматически из документа СоцФонд;

Документ СоцФонд_MU  содержит сведения об использовании средств социального страхования ЗЛ
в отчетном году по формам деклараций REV5,REV5a.
Документ формируется  из документа СоцФонд(автоматически) на основании заполненных документов Расчет Больничных
по кодам расходов: для одного ЗЛ количество строк соответствует количеству видов средств соц.страхования.

Корректировка данных любого из этих документов приводит к изменению данных в текущем документе.

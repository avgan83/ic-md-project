Документ предназначен для определения вида поступлений ОС по категориям собственности 
в целях налогообложения на основании приходных документов.
Порядок заполнения документа(шапка):
1.	Проставляем дату документа(изначально датой документа является рабочая дата);
2.	В ячейке "Основание" есть возможность охарактеризовать данное поступление, если такая необходимость существует;

Порядок заполнения документа (табличная часть):

3.	Документ заполняется путем нажатия в нижней части формы кнопки   
     "Заполнить". На основании данных о поступлении ОС по категориям  
     собственности в целях  налогообложения автоматически заполняется 
     табличная часть . В ячейке "Док Основание" появится документ по 
     которому ОС было оприходовано;
4.	Ячейка "Вид Поступления" заполняется вручную путем выбора из справочника "Виды поступлений ОС (в целях налогообложения)" форму поступления ОС .

Запись и проведение документа:
	Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем "Да" для записи его и "Нет" для отказа.
	При положительном ответе появится вопрос "Провести документ?" - ответ "Да" сформирует по документу проводки, 
	ответ "Нет" - проводки формироваться не будут но документ будет сохранен в базе и при необходимости к нему можно будет вернуться. 

Сохраненная информация будет использована в документе "Износ ОС в целях налогообложения".

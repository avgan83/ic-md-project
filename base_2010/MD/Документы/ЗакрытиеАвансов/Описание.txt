Документ предназначен для закрытия счета авансов 523.1 "Краткосрочные авансы выданные" на 221.1 "Счета к получению внутри страны" с одновременным 
формированием записей в книгу продаж.
Порядок заполнения документа:
1. Проставляем дату документа (изначально датой документа является рабочая дата);
2. Нажатие на кнопку "Заполнить" заполняет табличную часть документа с указанием контрагентов и сумм авансов по ним;
3. При окончании заполнения строки программа автоматически предложит заполнение следующей строки. При наличии дополнительных позиций ввод 
можно продолжить, при их отсутствии нажать на кнопку "Закрыть" в нижней части экрана, либо клавиши "Esc" - справочник будет закрыт, и Вы вернетесь в документ.
После чего надо будет удалить зарезервированную дополнительную строчку в документе, содержащую текущий номер строки без какой-либо дополнительной информации. 
Это осуществляется нажатием клавиши "Esc".
Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем "Да" для записи его в базу и "Нет" для отказа.
При положительном ответе появится вопрос "Провести документ?" - ответ "Да" сформирует по документу проводки, 
ответ "Нет" - проводки формироваться не будут, но документ будет сохранен в журнале и при необходимости к нему можно будет вернуться.
Кнопка "Закрыть" предложит Вам сохранить документ без формирования проводок по нему.

Данный документ предназначен для заполнения курсов сразу по нескольким валютам единожды, в
начале дня.
Документ позволяет одновременно заполнить курсы нескольких валют к молдавскому лею, 
не прибегая к возможности поочередного изменения этих значений через соответствующие истории
значений валют. Необходимо указать на какую дату будут записаны вводимые значения валют и,
соответственно, в табличной части указать вводимые валюты и проставить их курс.
Порядок заполнения документа:
1. Проставляем дату документа (изначально датой документа является рабочая дата);
2. Выбираем валюту по которой вводится курс;
3. Проставляем курс по данной валюте на дату заполнения документа;
4. Нажатие клавиши Enter является подтверждением правильности вноса информации по строке;
5. При окончании заполнения строки программа автоматически предложит заполнение следующей
строки. При наличии дополнительных позиций ввод  можно продолжить, при их отсутствии нажать
на кнопку "Закрыть" в нижней части экрана, либо клавиши "Esc" - справочник будет закрыт и вы 
вернетесь в документ.После чего надо будет удалить зарезервированную дополнительную строчку
в документе содержащую текущий номер строки без какой-либо дополнительной информации.
Это осуществляется нажатием клавиши "Esc".   
 Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем
"Да" для записи его в базу и "Нет" для отказа. При положи-тельном ответе появится вопрос 
"Провести документ?" - ответ "Да" запишет введенные курсы по каждой валюте в соответствующие
истории значений, ответ "Нет" - запись курсов формироваться не будет, но документ будет
сохранен в базе и при необходимости к нему можно будет вернуться.
Кнопка "Закрыть" предложит вам сохранить документ без записи курсов в истории.

Отчет "Взаиморасчеты" предназначен для учета взаиморасчетов между
контрагентами. 
Закладка "Основной" предлагает внести данные о:
- периоде отчета, который по желанию можно изменять с дату по дату
или, выбрав период за месяц, квартал и т.д.
- наименование клиента из справочника контрагентов; если позиция пустая, то
произойдет выборка по всем контрагентам,
- выборку можно сделать по операциям, а именно по поставкам, по покупкам, 
или по всем,
- выбрать кратный ил полный режим отражения отчета, 
- а также для удобства пользователя предлагается вывести отчетность
на русском или государственном языке.

Закладка "Счета" предлагает пользователю внести данные синтетических
 счетов, по которым будут произведены взаиморасчеты. ПРи желании можно 
 воспользоваться списком имеющихся счетов, или пополнить счета из 
 плана счетов, воспользовавшись серой кнопкой.
 Счета предлагаются в разрезе "по покупателям" или "по поставщикам".
 
 Кнопка "Сформировать" формирует отчетные данные по указанным позициям.
 Кнопка "Акт сверки" предлагает для анализа подробные данные взаиморасчетов 
 между контрагентами.

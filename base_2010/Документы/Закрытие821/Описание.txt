Документ предназначен для закрытия управленческих счетов 821 и 825.

Порядок заполнения документа (шапка): 
1. Проставляем дату документа (изначально датой документа является рабочая дата);
Внимание: Документ формируется только лишь после формирования документа "Расчет среднего процента торговой надбавки" и только на последнее число месяца.
2. В ячейке "По складу (отделу)" выбираем торговую точку (справочник "Склады") с которого были осуществлены продажа товаров. 
3. В ячейке "Комментарий" есть возможность охарактеризовать данное закрытие управленческих счетов. 
Порядок заполнения документа (табличная часть):
4. Нажатие кнопки "Заполнить" приводит к заполнению табличной части документа в разрезе ставок НДС по следующим позициям: остатки на конец месяца по счетам по счетам:
o 217.21 (Товар по розничным ценам);
o 821.21 (Торговая надбавка);
o 825.21 (НДС в рознице).
При некорректном заполнении документа для его быстрой очистки можно воспользоваться кнопкой "Очистить" которая полностью очистит табличную часть документа (заполнение шапки остается).
Печать:
Получение печатной копии документа осуществляется нажатием на кнопку "Печать". 
Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем "Да" для записи его в базу и "Нет" для отказа. При положительном ответе появится вопрос "Провести документ?" - ответ "Да" сформирует по документу проводки, ответ "Нет" - проводки формироваться не будут, но документ будет сохранен в базе и при необходимости к нему можно будет вернуться.
Кнопка "Закрыть" предложит вам сохранить документ без формирования проводок по нему. 

При проведении документ формирует следующие проводки:
Дт 825.21 Кт  217.21 сумма НДС с розничной суммы товара (если товар облагается НДС-процент НДС с Розничной суммы) в разрезе ставок НДС и складов
Дт 821.21 Кт 217.21 сумма торговой надбавки за минусом суммы НДС (Розничная сумма-процент НДС-себестоимость) в разрезе ставок НДС и складов.
После проведения документа счета "821.21" и "825.21" по торговым точкам и ставкам НДС должны быть закрыты!!!

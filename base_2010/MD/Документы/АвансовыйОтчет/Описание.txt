Документ предназначен для учета расчетов с персоналом предприятия по прочим операциям,
кроме расчетов по оплате труда. В авансовый отчет вносятся все расходы по подотчетному лицу
за отчетный период (по командировкам, на хозяй-ственные нужды и др.).         
Последовательность заполнение документа:
Закладка "Лицевая сторона":
1. Нумерация авансового отчета ведется автоматически. При необходимости нумерации документа
не с единицы (авансовые отчеты уже заполнялись вруч-ную) можно ввести следующий порядковый
номер. Дальнейшая нумерация будет продолжена с введенного вами номера;
2. Проставляем дату документа (изначально датой документа является рабочая дата);   
2. Выбираем счет подотчета (по умолчанию - 532.1);
3. Если выбран валютный счет то конкретизируем рабочую валюту по авансо-вому отчету.
Нажатием на кнопку "Валюта" открывается окно с возможность выбора конкретной валюты и ее 
курса на дату.
ВНИМАНИЕ: 
В программе существует возможность указания даты авансового отчета как даты составления
документа, а курса валюты на дату совершения хозяйственной операции подотчетным лицом. 
Для это используется окошко справа от выбранной валюты;
5. В ячейку "Сотрудник" - подотчетное лицо из соответствующего справочни-ка "Сотрудники";
6. В строку "Основания" вносится текст комментирующий хозяйственные операции по которым 
отчитывается сотрудник;
7. При условии, что выбранный сотрудник уже проходил по подотчету есть возможность просмотра
остатка или перерасхода денежных средств по нему. Для это необходимо нажать кнопку "Показать"
в окне "Предыдущий аванс". В ре-зультате будет выведен список всех кассовых документов
по которым сотрудник получал суммы в подотчет, а также итоговая сумма на выбранную дату: 
остаток или перерасход сумм.  
Закладка "Оборотная сторона":
8. Для переключения на оборотную сторону документа необходимо щелкнуть на закладку
находящуюся в левой верхней части документа с соответствующим названием;
9. В табличной части документа проставляется оправдательный документ подтверждающий 
совершение отдельной хозяйственной операции - для этого используется ячейка "Документ".
В ячейке "Дата" вносится дата которой датирован оправдательный документ. В ячейку
"Назначение" - комментарий по отдельно взятой хозяйственной операции, оформленной 
оправдательным документом;
10. Вносим корреспондирующий счет и аналитику по нему из соответствующих справочников.
Узнать о взаимосвязи счетов и аналитик (субконто) можно просмотрев
"План бухгалтерских счетов по НСБУ" находящийся в программе.
ВНИМАНИЕ:
В случае если к авансовому отчету приложен оправдательный документ без номенклатурной
конкретизации покупки (чек кассового аппарата), то в отчете используется счет ТМЦ напрямую
(211, 213, 217). Если же отпуск товара осуще-ствляется с оформлением товаротранспортной
накладной, то используется счет поставщика (521), а затем уже через документ  "Приходная ТТН"
приходуется товар;
11. Если по корреспондирующему счету ведется количественный учет - вводим количественное
выражение приходуемых ТМЦ;
12. Последней ячейкой к заполнению является ячейка для ввода суммы по данной хозяйственной
операции;
13. Нажатие клавиши Enter является подтверждением правильности вноса информации по строке;
14. По окончании заполнения строки программа автоматически предложит заполнение следующей
путем резервирования новой строки. При наличии дополнительных позиций ввод  можно
продолжить, при их отсутствии нажать на клавишу "Esc" - зарезервированная дополнительная
строка в документе содержащая текущий номер строки без какой-либо дополнительной
информации удалится.
Печать:
Получение печатной копии документа осуществляется нажатием на кнопку "Печать". 
Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем
"Да" для записи его в базу и "Нет" для отказа. При положительном ответе появится вопрос
"Провести документ?" - ответ "Да" сформирует по документу проводки, ответ "Нет" - проводки
формироваться не будут, но документ будет сохранен в базе и при необходимости к нему
можно будет вернуть-ся.
Кнопка "Закрыть" предложит вам сохранить документ без формирования проводок по нему.

Отчет "Анализ счета по субконто и периодам" сочетает в себе возможности 
двух стандартных отчетов: "Анализ счета по датам" и "Анализ счета по субконто".

Данный отчет позволяет получить информацию о том, с какими счетами за определенный 
период корреспондировал данный счет. В отличие от "Анализа счета по датам", 
"Анализ счета по субконто и периодам" позволяет получить остатки, обороты и обороты 
с другими счетами в разрезе субконто и с различной периодичностью (день, неделя, 
декада, месяц, квартал или год). 
С помощью данного отчета можно проанализировать, когда и с какой целью
были осуществлены операции по тому или иному субконто данного счета.

В верхней части диалога настройки отчета следует выбрать счет, по которому будет 
формироваться отчет и период, за который будут выдаваться бухгалтерские итоги.
Кроме того, следует выбрать один из вариантов периодичности отчета.

В зависимости от выбранного счета в диалоге предлагается установить режимы
разворота или отбора данных по субконто.

Если Вы хотите получить данные по каждому значению субконто отдельно, 
то переключатель соответствующий данному виду субконто следует установить 
в положение "Разворачивать". При этом если Вы выберете группу справочника,
то будут выведены итоги только по элементам выбранной группы.
Для включения в отчет самих групп следует включить флажок "Показывать группы".

Если Вы хотите получить данные только по одному конкретному значению субконто, 
то переключатель соответствующий данному виду субконто следует установить 
в положение "Отбирать", а в поле, предназначенном для выбора субконто, выбрать
конкретное значение. Если значение не выбрано будет сформирован отчет
по тем операциям, в которых использовалось пустое значение субконто.
Для выбора пустого значения следует нажать кнопку "Х".

Если вид субконто не должен никак использоваться в отчете, то следует отключить
флажок "Учитывать".

С помощью кнопок со стрелками Вы можете изменять порядок расположения 
субконто в диалоге и соответственно управлять вложенностью итогов в отчете.

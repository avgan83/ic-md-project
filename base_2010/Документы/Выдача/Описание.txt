Данный документ предназначен для автоматического расчета сумм подоходного налога,
пенсионного фонда, профсоюзного фонда с последующие удержанием их из заработной платы
сотрудников.
Последовательность заполнение документа:
Существует 2 варианта заполнения документа:
- автоматическое заполнение документа на основании ранее введенного документа по начислению 
заработной платы. Это возможно находясь в журнале "Заработная плата", выделив курсором 
документ по начислению заработной платы на основании которого заполняется выдача и нажатием
пиктограммы вверху рабочего окна программы.
- ручное заполнение:
1. Проставляем дату документа (изначально датой документа является рабочая дата);
2. Выбираем счет подоходного налога (по умолчанию - 534.7) и аналитику по нему из справочника
"Налоги";
3. Выбираем счет пенсионного фонда (по умолчанию - 533.1) и аналитику по нему из справочника
"Налоги";
4. Выбираем счет кассы (по умолчанию - 241.1) и аналитику по нему из справочника
"Денежные средства";
5. Выбираем счет профсоюзного фонда (по умолчанию - 533.1) и аналитику по нему из справочника
"Затраты";
6. Дальнейшее заполнение ячеек происходит автоматически на основании информации введенной 
в константах.
7. Нажатие клавиши Enter является подтверждением правильности вноса информации по строке;
8. При окончании заполнения строки программа автоматически предложит заполнение следующей 
строки путем открытия справочника "Сотрудники" для выбора новой аналитики. 
При наличии дополнительных позиций ввод  можно продолжить, при их отсутствии нажать
на кнопку "Закрыть" в нижней части экрана, либо клавиши "Esc" - справочник будет закрыт и вы 
вернетесь в документ. После чего надо будет удалить зарезервированную дополнительную
строчку в документе содержащую текущий номер строки без какой-либо дополнительной
информации. Это осуществляется нажатием клавиши "Esc".
Печать:
Документ может формировать три вида печатных форм (при нажатии на кнопку "Печать"): 
1.	Платежная ведомость по всем сотрудникам
2.	Распечатка расчетного листка по каждому сотруднику.
3.	Расчетная ведомость по всем сотрудникам.  
Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем
"Да" для записи его в базу и "Нет" для отказа. При положительном ответе появится вопрос
"Провести документ?" - ответ "Да" сформирует по документу проводки, ответ "Нет" - проводки
формироваться не будут, но документ будет сохранен в базе и при необходимости к нему можно
будет вернуться.
Кнопка "Закрыть" предложит вам сохранить документ без формирования проводок по нему.

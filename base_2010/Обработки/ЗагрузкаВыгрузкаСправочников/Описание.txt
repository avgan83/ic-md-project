	Эта универсальная (т. е. работающая в любой конфигурации) обработка, которая 
позволяет выгружать в текстовый формат структуру и содержимое выбранных справочников. 
А также, загружать справочники данными из текстового файла. Обработка позволяет 
переносить данные из разных конфигураций и справочников. При этом соответствие 
реквизитов справочников и режим загрузки устанавливается пользователем.
	Для того, чтобы выгрузить данные из справочников, нужно указать какие 
справочники и их реквизиты будут выгружаться, а также имя и путь, к файлу, в который 
будут выгружены данные.
	Для указания подлежащих выгрузке справочников надо выбрать закладку "Выгрузка" 
и в списке отметить требуемые справочники. Нажав кнопку "Реквизиты" пользователь может 
выбрать выгружаемые реквизиты текущего справочника. Для выполнения выгрузки следует 
указать имя файла, в который нужно выгрузить данные, и нажать кнопку "Выгрузить". В 
процессе выгрузки будет создан файл с указанным именем, содержащий выгруженные данные 
в специальном формате.
	Для загрузки данных в справочники,  на закладке укажите "Загрузка" файл, в 
котором они содержатсяи нажать кнопку "Анализ". При этом будет заполнена таблица 
соответствий справочников и таблицы соответствий их реквизитов. В колонке "Справочник. 
загруж." содержится набор справочников полученных из указанного файла загрузки. В колонке 
"Справочник. текущ." соответствующие им справочники из текущей конфигурации, в которые 
будет загружена информация. Соответствие между загружаемыми и текущими справочниками 
устанавливается по идентификаторам справочников. При этом пользователь сам может указать 
соответствие между справочниками. Для этого выбрать значение в колонке "Справочник. текущ." 
(клавиша "Enter", или двойной щелчок мыши) и из предложенного списка выбрать 
соответствующий справочник или <не загружать>.  Те справочники для которых не указано 
соответствие  загружаться не будут. 
	Для установки соответствий между реквизитами загружаемого и текущего 
справочника нужно нажать кнопку "Реквизиты". В колонке "Рекв.текущ." список реквизитов 
текущего справочника, в колонке "Рекв.загруж." соответствующие реквизиты загружаемого 
справочника, или значение которое должен принимать данный реквизит элемента 
справочника. Здесь же для подчиненных справочников можно указать какому владельцу 
подчинять загружаемые элементы если для них не найден соответствующий владелец.
	На закладке "Параметры загрузки" можно указать дополнительные условия загрузки. 
пользователь может указать как разрешать конфликты связанные с нарушением уникальности 
кодов справочника или вообще запретить синхронизацию по коду. Если пользователь
запретит синхронизацию по коду, элементы справочника всгда будут добавляться с новым кодом.
В случае если разрешена синхронизация по коду, у пользователя есть три варианта 
разрешения конфликта дублирования кодов:
1.	Не загружать элемент - соответствующий элемент не будет загружен и об этом будет 
	выдано сообщение.
2.	Исправлять существующий элемент - реквизиты существующего элемента будут загружены 
	новыми данными.
3.	Добавлять новым кодом - будет создан новый элемент, и ему присвоен новый код
	Так же на данной закладке пользователь может указать что справочники в которые 
будет происходить загрузка должны быть очищены (все элементы помечены на удаление либо 
удалены).
	Для выполнения загрузки, после указания всех параметров загрузки и настройки 
соответствий  справочников и их реквизитов, нужно нажать кнопку "Загрузить".
	Настройки элементов формы обработки можно сохранять, формируя списки настроек
отдельно для выгрузки и загрузки справочников. Для сохранения настроек необходимо 
перейти на закладку "Настройки". Если нет ни одной сохраненной настройки, в поле "Настройки" 
появляется наименование по умолчанию - "Основной". Это наименование можно изменить и нажать 
на кнопку "Сохранить". Тогда в список настроек добавится наименование сохраненной настройки. 
	Для восстановления элементов нужной настройки, необходимо выбрать эту настройку 
из списка и нажать на кнопку "Восстановить". Для удаления сохраненной настройки 
необходимо выбрать эту настройку из списка и нажать на кнопку "Удалить".

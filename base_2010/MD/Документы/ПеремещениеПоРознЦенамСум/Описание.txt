Документ предназначен для передачи товаров из одной торговой точки в другую по розничным ценам.
Порядок заполнения документа (шапка): 
1. Проставляем дату документа (изначально датой документа является рабочая дата);
2. В ячейке "Расход" выбираем торговую точку (справочник "Склады") из которой будет перемещаться товар по розничным ценам. 
3. В ячейке "Приход" выбираем торговую точку (справочник "Склады") на которую будет перемещаться товар по розничным ценам. 
4. В ячейке "Комментарий" есть возможность охарактеризовать данное перемещение товара между торговыми точками, если такая необходимость существует. 
Порядок заполнения документа (табличная часть):
5. Табличную часть можно заполнять двумя различными способами, выбор которых полностью зависит от бухгалтера и особенностей перемещения товаров между торговыми точками
6. При использовании любого из способов заполнения документа ячейки "Остаток", "Кол. остатка", "Количество",  "Ед. изм", "ЦенаСпр", "СуммаСпр" будут заполнены автоматически.
7. В обоих случаях после заполнения табличной части пользователь должен по необходимости откорректировать (ячейка "Количество") количество перемещаемого товара в разрезе номенклатуры. Ячейка "СуммаСпр" будет автоматически заполнена путем перемножения количества перемещаемого товара и розничной цены за единицу.
8. Нажатие клавиши Enter является подтверждением правильности вноса информации по строке.

Печать:
Получение печатной копии документа осуществляется нажатием на кнопку "Печать". 

Запись и проведение документа:
Сохраняем документ путем нажатия на кнопку "ОК". На вопрос "Сохранить документ?" отвечаем "Да" для записи его в базу и "Нет" для отказа.
При положительном ответе появится вопрос "Провести документ?" - ответ "Да" сформирует по документу проводки, 
ответ "Нет" - проводки формироваться не будут, но документ будет сохранен в базе и при необходимости к нему можно будет вернуться.
Кнопка "Закрыть" предложит вам сохранить документ без формирования проводок по нему. 

При проведении документ формирует следующие проводки:
Дт 217.21 Кт 217.21 сумма себестоимости  и количество по передаваемому товару (сумма учета) в разрезе ставок НДС и складов
Дт 825.21 Кт 825.21 сумма НДС с розничной суммы товара (если товар облагается НДС-процент НДС с Розничной суммы) в разрезе ставок НДС и складов
Дт 821.21 Кт 821.21 сумма торговой надбавки за минусом суммы НДС (Розничная сумма-процент НДС-себестоимость) в разрезе ставок НДС и складов
